Debug a generated language processor at the source level

Eli request: 
    xyz.specs : dbx

Executes the language processor unter the control of the source level debugger specified in environment variable ELI_DEBUGGER.

Eli request: 

    xyz.specs : +debug : exe>.
generates the language processor with debugging information turned on (C compiler option "-g").

Eli request: 

    xyz.specs : source > xyz.src
generates the language processor's sources. These can be compiled outside of Eli with debugging information turned on to allow for source level debugging, e.g.: 
   cd xyz.src;
   make CC="cc -g"
   workshop xyz.exe
