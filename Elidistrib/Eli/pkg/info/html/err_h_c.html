User supplied .h and .c specifications

User supplied C-modules have to be mentioned in some .specs file. Furthermore one or several .head files or .HEAD.phi files have to be provided to make user declarations available in the attribute evaluator.

Typical Error Situations

  • C errors:

    ==> Compiler Messages, or Runtime Errors

  • Missing .head or .HEAD.phi entries:

    ==> Compiler Warnings (missing declarations), linker errors (undefined symbol).
