Reference Manuals

This section of the Eli documentation contains reference manuals for various notations used by the Eli system.

The User Interface Reference contains information on the user-level commands used to ask Eli to produce various language processing products.

The Products and Parameters Reference provides details on the products that may be requested and the parameters that can be used to modify the generation of the products.

The LIDO Reference describes in detail the LIDO languauge that is used in Eli to specify computations in trees.
