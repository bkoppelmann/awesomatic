Translation Tasks

This section of the Eli documentation addresses the typical translation tasks that are supported by the Eli system.

The Lexical Analysis manual describes how to describe patterns of characters in the input text so that they may be recognised and processed.

The Syntactic Analysis manual describes how to specify the syntactic structure of the input text so that it may parsed automatically and the resulting structure represented as a tree.

The Computations in Trees manual shows how to use the LIDO language to specify sophisticated processing of tree-structured information.
